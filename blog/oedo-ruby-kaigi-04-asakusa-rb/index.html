言語の勉強法

学校や資格は、基礎は学べるけどやればやるほど効果が薄くなる。語学は練習することが一番大切。

より大切なことは文化を知り、コミュニティに飛び込んでいけるようになること。

1) ENCOUNTER => ネット文化に触れる
2) EMBARGE => 自ら飛び込む
3) LEARN/ENJOY => 楽しみながら学ぶ

英語を学習する際のオススメのアクション！

1) REDDITを見る => TIL、AIM、ELI5(EXPLAIN LIKE I'M 5)がオススメ
2) ネタの宝庫、わからなければググる
3) コミュニティでネタをわかるようになる
