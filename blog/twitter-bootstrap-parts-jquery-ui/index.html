ひと味ちがうTwitter Bootstrapリソースまとめ、UIパーツ編

Aug 19th, 2013

  • Tweet

Twitter-bootstrapのUIパーツをまとめたページです。素のTwitter Bootstrapにはない、かゆいところに手が届きそうなUIパーツをまとめています。

姉妹ページの『Twitter Bootstrapについてのテンプレート(レイアウト)のまとめ』もどうかよろしくおねがいします！

この記事は今後も新しいリソースを見つける度に順次アップデートしていく予定です！

(08/20 21:00) Fuel UXを追加

Fuel UX

Fuel UXはTwitter Bootstrapのウィジェットをまとめて拡張してくれるツールです。チェックボックスや、コンボボックス、データグリッド、ピルボックス、検索ボックス、スピナー、エクスプローラーっぽいツリー、ウィザードなどが追加されます。

MoonGiftさんの記事『Bootstrapで物足りなく感じているウィジェットをまとめて提供「Fuel UX」|オープンソース・ソフトウェア、ITニュースを毎日紹介するエンジニア、デザイナー向けブログ』にはサンプル画像付きで表示されているのでこちらもオススメです！

LayoutIt!

ドラッグアンドドロップ形式でBootstrapペースのレイアウトを作成できるWebツールです。作成したら、HTML/CSSでダウンロード出来ます！

Bootsnipp.com

BootstrapのUIパーツを集めたサイトです。テンプレートよりも簡単に導入できるので良かったら使ってみてください！

Twitter Bootstrap templates

Twitter-Bootstrapのためのテンプレート集です。無料のものと有料のものがあるようです。

Pines Notify

下の画像の右上に出ているような、javascriptによるノーティフィケーションを表示してくれるツールです。Webサイト上でインタラクティブなお知らせを表示する場合などに活用できそうです！

bootstrap-wysihtml5

BootsrapにマッチしたWYSIWYGを1行で追加できるUIパーツです。

Datepicker for Bootstrap

Bootsrapにマッチしたデータピッカーです。

bootstrap-datetimepicker

Bootstrapにマッチしたデータピッカー、第２段です。

jQuery prettyCheckable

ラジオボタンやチェックボックスを写真のようなデザインに置き換えてくれるjQuery プラグインです。コンバージョンを意識する場合には選択肢としてありだと思います！

Bootstrap Form Helpers

Bootstrapに最適なフォーム入力支援jQuery Pluginです。
サポートしてくれるフォームのタイプは12種類です。

(1) セレクトボックス
(2) 国情報
(3) 州(都道府県)
(4) 通貨
(5) 電話番号
(6) 言語
(7) タイムゾーン
(8) Googleフォントの種類
(9) Date Picker(日付)
(10) 時刻
(11) フォント種類
(12) フォントのサイズ

MagicSuggest

BootstrapにマッチしたオートコンプリートのjQuery Pluginです。

Select2

検索時の補完や、タグの複数選択などができるjQuery Pluginです。後発のツールだけあって、その分多機能です！

Bootstro.js

Bootstrapにマッチしたサイトの利用手順をオーバーレイで表示してくれるjQueryです。

Twitter Bootstrap Button Generator

Bootstrapの基本デザインをベースにしたボタンを生成できます。ベースの色合いを崩さないので、とっても使いやすそうです！

Bootstrap Tags

フォーム内でのタグの選択などに最適なjQuery Pluginです。

Beautiful Buttons for Twitter Bootstrappers

BootstrapにマッチしたCSSボタンを作ることが出来るツールです。色合いやグラデーションを簡単に調整することができます。こちらはBootsrapを拡張してオリジナリティのあるボタンが出来ます。

Social Buttons for Twitter Bootstrap

Bootstrapのデザインにマッチしたソーシャル系のボタンを作成出来ます。
(Facebook, Twitter, GitHub, Pinterest, Linkdinなど)

Bootstrap Image Gallery Demo

Bootstrapのデザインにマッチしたイメージ・ギャラリーを作成することができます。

jQuery File Upload

jQueryによるFile アップローダーです。コレを使うメリットは、アップロード前のプレビュー表示や、プログレスバーの表示、ドラッグアンドドロップへの対応などなど。

jquery-simplecolorpicker

サイトの種類次第では地味に必要になるカラーピッカーのTwitter-Bootstrap版。自分で作りこむと結構骨が折れそうなのでオススメです！

jQuery-menu-aim

これはBootstrap用ではないですが、狙いは面白いのでご紹介。Amazonのメニューっぽいドロップダウンメニューを提供していくれるjQueryプラグインです。

BootstrapCDN

Twitter Bootstrapを配信してくれるCDNです。CDNを使ってサーバ負荷分散をしたい場合には選択肢としてありだと思います。
