😼 Herokuユーザー登録

Herokuでユーザー登録をします。
アカウント作成時に使ったメールアドレスとパスワードは後で使うのでメモっておいてください。

Heroku Toolbeltをインストールします。Heroku Toolbeltの機能は；

  • Heroku client - Heroku appsを管理するためのCLI
  • Foreman - ローカルでHerokuアップを動かすためのオプション
  • Git - HerokuへのPush/ソースのバージョン管理
