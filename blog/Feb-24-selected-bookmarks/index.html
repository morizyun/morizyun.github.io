2/24 厳選エンジニア向けブックマーク

Feb 24th, 2014

  • Tweet

2014/02/24に気になったブックマークの一覧です。

新サービス

無料パフォーマンステスト | 負荷テスト
URLだけわかれば、後は自動で負荷テストを行ってくれるサービス。これはかなり面白そう！

Gitterが一般βになったので早速やってみたんだけど、ムッチャ感動するレベルだった
GitHubのリポジトリやOrganizationごとにチャットルームを作成することができるツール『Gitter』の紹介記事。そのリポジトリの活動状況をチャットルームに簡単に表示することができるそうです。

無料・電子書籍

Rails AntiPatterns
全311ページに及ぶ、Railsリファクタリングのベスト・プラクティスがマトメられた無料の電子書籍です。英語ですが、読む価値がある素晴らしいコンテンツです！

Objective-C / iOS

iOSの開発をする上で絶対に使うべき！外せない！webサービス、開発ツール集 - Qiita
殿堂入りレベルのObjective-C / iOSに関する記事。素晴らしい！

Javascript

ソーシャル投稿ボタンのクリックのcallbackをJS から知る方法
自分のサイトに設置したTwitter/Facebook/Google+などのボタンから投稿した後のCallbackを取得するための方法のマトメ。Google Analyticsでtrackingする方法や、Facebookクローラについての記述もあり。

CUI ツール

tigでgitをもっと便利に！ addやcommitも
CUIでgitを操作する際に欠かせないツール『tig』のTips的な記事。
