Objective-C CoreDataのバージョンアップ in XCode 4.5.2[iPhone]

Nov 30th, 2012

  • Tweet

XCode 4.5.2でCoreDataのEntity(エンティティ)やAttribute(アトリビュート)の定義を変更した場合のバージョンアップの手順です。

バージョンアップ手順

XCode 4.5.2のメニューバーの “Editor” => “Add Model Version…” を選択。

新しいバージョンの名前と、ベースになるバージョンを選択。

新しいバージョンがCore Dataの配下に出来ます。

新しいバージョンのEntity(エンティティ)やAttribute(アトリビュート)を変更します。

わかると本当に簡単な作業ですが、うまく検索ワードを見つけられずに随分ハマりました。

もっといい方法やわかりやすい説明、間違い等あれば連絡下さい！熱烈大歓迎です！
