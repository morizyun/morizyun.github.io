ひと味ちがうTwitter Bootstrapの18個の無料テンプレート&有料まとめサイト

Aug 19th, 2013

  • Tweet

Twitter Bootstrapはデザインが苦手なプログラマのための必須ツールです。今回は、一味違ったBootstrapサイトを作るときにきっと参考になるテンプレートをまとめてみました！

追加情報

最新のBootstrap3に対応したテーマの一覧です。最近はこちらのほうがオススメかも！
Bootstrap3 の無料テーマ・格安有料テーマまとめのまとめ

BootstrapをRails4.1に簡単に導入する手順です。RailsでBootstrapを使う場合はこちらもオススメです！
Bootstrap/BootswatchをRails 4.1に導入！ - 酒と泪とRubyとRailsと


無料のテンプレート

無料のBootstrapテンプレートの紹介です。BootswachのようにCSSだけで適用できるものと、HTML/CSS/JSなどいろいろ追加しないと実現できないものがありますが、Bootstrapを使ったサイトを作るときには参考になると思います！

Bootswatch

このテンプレートネタを扱う上で、まず第一に語るべきなのが鉄板の「Bootwatch」です。

ちなみに、BootwatchをRailsで使う手順を「RailsにMetroライクなBootstrapテーマCosmoを速攻導入する」にまとめています。よかったら参考にして下さい！

Flat-UI

Twitter BootstrapベースでつくられているフラットなデザインのUIキットです。ベクター素材のカッコイイアイコンまで含まれています！

Bootstrap Themes

管理画面に最適なBootstrapのテーマです。バランスもよくってカッコイイです。リンクを付けたかったり、自分のサーバ以外で使いたい場合には有料みたいです。

Flatstrap

Twitter Bootstrapをフラットなデザインにしたテーマです。カッコイイ！

Andia

クオリティの高いレスポンシブ・デザインに、「Home, Portfolio, Services, About, Contact」までデザイン済という至れり尽くせりのテンプレートです。

TODC Bootstrap

Twitter Bootstrapからフォークして、GoogleっぽいUIにしているテンプレートみたいです。

GreenMind

クラスメソッドの山田さんが制作されたテーマです。シックな色合いが抜群に素敵ッス！

Respond 1.5

落ち着いた色使いで、シンプルなので使いやすいテーマです。

BootMetro

Windows 8の MetroっぽいUIを作ることが出来るBootstrapのテンプレートです。

Bootbusiness

ビジネス、会社向けのWebデザインに最適なTwitter-Bootstrapテンプレートです。

KataKlimt

Bootstrapでここまでできるのか？って思えるほどすごい出来です。

Stylish

これもクオリティ高いです。黒を貴重にして「スタイリッシュ」なデザインになってます。

名前不明

シンプルですが、綺麗なデザインです。本のようなページ遷移があって面白いです。(ページ遷移はBootstrapとは直接の関係はないですがw)

JessicaWhite

白を基調にしたデザインです。配置のバランスも抜群です！

Charisma

管理画面のテンプレートです。ところどころに小さな動きがある点と、Bootswatchベースのテーマが9種類ある点がポイントです！

FBStraap

Facebook風のデザインを再現できるテンプレートです。Facebookアプリに最適かも。

Geo for Bootstrap

ネタなテーマです。ほんの少し前まではこんなサイトが乱立してたよなぁww

Coming soon page in Twitter Bootstrap

よくある告知系のページをBootstrapで作る場合に使えそうなツール。動きがあってデザインがきれいなページを簡単に作れそうです。

1pxdeep

デザイ系有名なブログコリスさんの記事『Bootstrap 3対応、高性能なカラースキームツールを備えたテーマファイル-1pxdeep』で紹介のあったツールです。自分の好みの色を指定すると、Bootstrap 3対応のテーマを生成してくれるツールです。個人でも商用でも無料で利用できます。

Themestrap

Bootstrap 3に対応した無料のテーマを集めているサイトのようです。2013/08/30時点ではまだ２つですが、なかなかセンスの良いテーマが集まっています！

BootTheme Gallery

Bootstrapのテーマをユーザーが作成できるツールBootThemeを使って生成されたテーマのギャラリーです。玉石混交ですが、たまに面白い配色のデザインがあります！

有料テンプレートサイト

ここからは有料のテンプレートの紹介です。メインの価格帯は$20(1,800円)くらいなので、作業時間の節約や複数のサイトで使いまわせる点で有料テンプレートも選択肢としてありだと思います！

WrapBootstrap

Bootstrapのテーマを専門に扱っているサイトです。おそらくいちばん有名なサイトだと思います。テーマの数と質がどんどん上がってきているので、見るだけでもオススメです！

Bootstrap | ThemeForest

Wordpressのテーマの販売で有名なThemeForestでもBootstrapのテーマが販売されています。こちらもなかなかの数です。

Premium Twitter Bootstrap Themes

$5(500円弱)からBootstrapのテーマが販売されています。

おまけ1: Bootstrap Expo

Bootstrapで作られているサイトの中でも、特にイケてるサイトが厳選されています。あの有名サービスもBootstrapベースなのか？って驚きました〜＾＾

このテーマに関連するオススメ本

『インタフェースデザインの心理学 ―ウェブやアプリに新たな視点をもたらす100の指針』はデザイナさんだけでなく、プログラマーさんや一般のビジネスマンさん、管理職の方が見ても「学び」が大きい本です。「人の認知の仕方、感じ方、注意の向け方など」をビジネスに結びつけて書いているのでとても読みやすいです！
