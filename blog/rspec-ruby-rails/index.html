(4) matcher（マッチャ）

be_true のようなメソッドを matcher（マッチャ）といいます。 Stack クラスのRSpecに3種類のmatcherを追加します。

ここで使われているマッチャは次の通りです。

"#size"は「==」
"#pop"は「raise_error」
"#push"は「change」

結果はこちら。

rspec -fs -c stack_spec.rb 

Stack
  新規作成の場合
    empty?
      should be true
  空の場合
    #size
      should == 0
    #pop
      should raise Stack::EmptyError
    #push
      should change #size

Finished in 0.00388 seconds
4 examples, 0 failures
