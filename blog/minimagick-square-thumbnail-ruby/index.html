RubyでMiniMagickを使って正方形のサムネイルを作成

2017-01-17
  • Tweet

ImageMagicのRubyラッパ Minimagick
を使って縦横の比率がバラバラの画像から、正方形のサムネイルを切り出すRubyのサンプルソースです。


🗽 Installation

Gemfileに以下を追加して bundle install を実行してください。

🐠 ソースコード

サンプルソースでは次のことをしています。

  • (1) サンプルの画像をダウンロード
  • (2) MiniMagickを使って正方形のサムネイル画像を切り出し
  • (3) 画像をファイルに保存

では動作確認です。

上の画像(640×480)が下のように正方形(300x300)になりました。
